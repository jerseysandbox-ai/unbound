# Unbound 🌱

**AI-powered daily lesson plan generator for neurodivergent homeschoolers.**

A parent fills in a short profile about their child, pays $9, and receives a personalized full-day lesson plan generated by a team of specialized AI agents — designed around how *their* kid learns best.

---

## How the Agent Team Works

Plan generation runs in 3 rounds:

**Round 1 — Sage (sequential)**
> 🧠 Sage: Neurodivergent/PANS specialist. Reviews the full child profile and produces a "Child Brief" — key sensitivities, engagement strategies, and pacing notes that all other agents factor in.

**Round 2 — Subject Specialists (parallel)**
> Each receives the child profile + Sage's Child Brief and designs one activity for their subject:
- 📐 Euler — Math
- 📖 Darwin — Science
- ✍️ Paige — Language Arts
- 🌍 Atlas — Social Studies
- ❤️ Grounded — SEL + Executive Functioning
- 🎨 Studio — Arts & Creative Expression
- 💡 Spark — Entrepreneurship

**Round 3 — Architect (sequential)**
> 📋 Architect: Takes all 7 subject outputs and assembles them into one warm, cohesive daily plan with pacing, transitions, and encouraging framing.

## Tech Stack

- **Next.js 15** (App Router)
- **Vercel** hosting + **Vercel KV** (Redis) for plan storage (24h TTL)
- **Stripe** one-time payment ($9)
- **Anthropic Claude claude-sonnet-4-6** for all agents
- **Cloudflare Turnstile** for bot protection
- **@react-pdf/renderer** for PDF export

---

## Routes

| Route | Purpose |
|-------|---------|
| `/` | Landing page |
| `/profile` | Child profile form |
| `/checkout` | Stripe payment |
| `/plan/[id]` | View + download plan |
| `POST /api/create-payment-intent` | Create Stripe PI (verifies Turnstile) |
| `POST /api/generate-plan` | Run agent pipeline + store in KV |
| `GET /api/get-plan/[id]` | Retrieve plan from KV |

## Local Development

```bash
npm install
cp .env.local.example .env.local
# Fill in env vars
npm run dev
```

For local KV, you can use the `@vercel/kv` local emulation or point to a real Upstash Redis instance.

---

## Deployment (Vercel)

1. Import repo at vercel.com
2. Add a **KV Store** (Vercel Integrations → Upstash Redis)
3. Set all env vars listed above
4. Deploy — that's it
5. Add your Vercel domain to the Cloudflare Turnstile widget settings
